TIPOS DE ETIQUETAS

En HTML y CSS tenemos diferente tipos de etiquetas cada uno con un propósito en particular

Una etiqueta se puede componerse de dos maneras:

ESTRUCTURA DE ETIQUETAS
Etiquetas que solo aperturan Etiquetas que tinen apertura y cierre

Lista de etiquetas dirigidas al texto

⇝ Títulos

Un título es exactamente lo que parece: un título o un subtitulo real que encabezan los temas. Los títulos pueden tener dimensiones entre 1 y 6, siendo 1 la más grande y 6 la más pequeña.

Podemos crear títulos tanto para el body como para el head.

HEAD

  • < title >< /title > Permite crear un título que se visualizara en la pestaña del navegador, el cual puede ser nombre del creador. Ejemplo "Dulce"

BODY

  • < h1 - /h1 > etiqueta que se utiliza para colocar el título pricipal de nivel 1
  • < h2 - /h2 > etiqueta que se utiliza para colocar títulos de nivel 2
  • < h3 - /h3 > etiqueta que se utiliza para colocar títulos de nivel 3
  • < h4 - /h4 > etiqueta que se utiliza para colocar títulos de nivel 4
  • < h5 - /h5 > etiqueta que se utiliza para colocar títulos de nivel 5
  • < h6 - /h6 > etiqueta que se utiliza para colocar títulos de nivel 6

⇝ Párrafos

Un párrafo se abre y se cierra con la etiuqueta "p". Para crear varios párrafos tenemos que abir esta etiqueta las veces que lo necesitemos para cambiar de uno a otro.

BODY

  • < p - /p > etiqueta que se utiliza para crear párrafos. Estos pueden veriar de tamaño dependiendo de la información que insertemos en el contenedor.

⇝ Listado y viñetas

Las etiquetas utilizadas para crear un listado seran ol y li las cuales las solemos para una enumeración de cosas, personas, cantidades, etc.

Para las viñetas utilizaremos la etiqueta ul

BODY

  • < ol - /ol > etiqueta que se utiliza para crear listas ordenadas
  • < li - /li > etiqueta que se utiliza para crear listas no ordenadas
  • < ul - /ul > etiqueta que se utiliza para crear viñetas a las listas

Lista de etiquetas dirigidas a tablas

⇝ Tablas

Las tablas son estructuras útiles y a menudo fáciles de interpretar para relacionar datos e información de manera pertinente. Y para crearlas necesitamos de las siguientes etiquetas, siendo la principal "table", la cual va a tener columnas y filas.

BODY

  • < table - /table > etiqueta que se utiliza para definir que crearemos una tabla
  • < tr - /tr > etiqueta que se utiliza para crear filas en una tabla
  • < td - /td > etiqueta que se utiliza para crear columnas en una tabla

ATRIBUTOS DIRIGIDOS A LA ETIQUETA TABLA

  • border = " " atributo que se utiliza para editar y definir los bordes de una tabla, ya sea aumentandolos o eliminandolos
  • cellspacing = " " atributo que se utiliza para definir el espacio que hay entre los bordes de la tabla
  • cellpadding = " " atributo que se utiliza para definir el espacio entre los bordes de la celda y el contenido de la misma

ATRIBUTOS DIRIGIDOS A LA ETIQUETA <TD>

  • colspan = " " atributo que permite unir una celda con las celdas contiguas. Indica la cantidad de celdas unidas en horizontal

Lista de etiquetas dirigidas a objetos

⇝ Estructuras

Estas etiquetas pueden ser redimensionadas tanto en ancho y como en alto. A estas etiquetas también las conocemos como cajas.

BODY

  • < header - /header > etiqueta que puede contener algunos elementos de encabezado, así como también un logo, un formulario de búsqueda, un nombre de autor y otros componentes.
  • < nav - /nav > etiqueta que sirve para contener los elementos de navegación, ya sean links , menús o cualquier otro elemento.
  • < div - /div > Sirve para crear secciones o agrupar contenidos. Está definido como: Elemento en bloque.
  • < section - /section > etiqueta que sirve para representar una parte de un documento o artículo. Define las sección en un documento
  • < article - /article > etiqueta que es un contenedor de bloques de contenido que se consideran independientes del sitio web, nos ayuda a definir un contenido específico dentro de una web y que por si solo tiene sentido.
  • < footer - /footer > representa al pie de una sección o documento, donde los autores habitualmente colocan firmas, información acerca del autor, información de licencias, documentos relacionados, etc.
  • < img > etiqueta que permite añadir atributos que controlará el comportamiento y la apariencia de una imagen en una página.

ATRIBUTOS DIRIGIDOS A LA ETIQUETA < img >

  • src = " " atributo que contiene como valor un url. Este url indica la dirección y el nombre de la imagen que se desea insertar. La imagen puede estar en cualquiera de los formatos que reconocen los navegadores (jpg, gif, png, etc.)

Lista de etiquetas dirigidas a enlaces de hipertextos

⇝ Hipertextos

Son el elemento característico de la web porque permiten que las páginas se relacionen entre sí. La sencillez y libertad con que se pueden enlazar páginas situadas en servidores distintos son el motivo del éxito de la web.

BODY

  • < a - /a > etiqueta que nos ayuda a poder crear un enlace a una página web

ATRIBUTOS DIRIGIDOS A LA ETIQUETA < a >

  • href = " " atributo que permite que pongamos el enlace de lo que queremos conectar.

Lista de etiquetas dirigidas a formularios

⇝ Formularios

Su propósito es el de recolectar información proporcionada por los visitantes del sitio, la cual es luego enviada nuevamente al servidor

BODY

  • < form - /form > etiqueta que representa una sección de un documento que contiene controles interactivos que permiten a un usuario enviar información a un servidor web
  • < label - /label > etiqueta que representa un título para un elemento en una interfaz de usuario
  • < input > etiqueta que se usa para crear controles interactivos para formularios basados en la web con el fin de recibir datos del usuario.
  • < button - /button > etiqueta que crea botones marcadores. Representa un elemento cliqueable de tipo botón

Conectar HTML con CSS

⇝ ETIQUETA A UTILIZAR

Para conectar un css a html utilizamos la etiqueta - link rel="stylesheet" href="css/css.css" -

Dentro de href debemos escribir el nombre la carpeta donde guardamos nuestro css a utilizar seguido de / y nuestro css.

Listado de propiedades generales de CSS

Su función es definir la estética o la presentación de una página web en general

GENERALES

  • margin: define los margenes externos de un elemento
  • padding: definen los margenes internos de un elemento

Listado de propiedades dirigidas al texto

Nos permiten controlar el texto como bloque, es decir, afectan al interlineado, a la separación entre palabras, al tabulado, color, fuente, etc

TEXTO

  • color: define el color del texto
  • font-size: define el tamaño del texto
  • font-family: define la tipografía del texto
  • text-aling: define la posición del texto

Listado de propiedades dirigidas a cajas

Todo en CSS tiene una caja alrededor, y comprender estas cajas es clave para poder crear diseños con CSS o para alinear elementos con otros elementos.

CAJAS

  • background: define el color de relleno de un elemento
  • width: define el ancho de un elemento
  • heigth: define el alto de un elemento
  • border: define bordes a una imagen, color y grosor
  • border-radius: define el dobladillo del borde
